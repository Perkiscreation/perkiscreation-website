INSTALLATION:
1. Server mit Strg + C stoppen.
2. ZIP entpacken.
3. Alles kopieren nach C:\\Users\\timot\\perkiscreation-website
4. Dateien im Ziel ersetzen.
5. Falls vorhanden: postcss.config.mjs löschen.
6. VS Code neu öffnen.
7. Terminal -> New Terminal.
8. npm install
9. npm run dev
10. http://localhost:3000 öffnen.
