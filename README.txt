INSTALLATION:
1. Server mit Strg + C stoppen.
2. ZIP-Datei entpacken.
3. Im entpackten Ordner alles mit Strg + A markieren und mit Strg + C kopieren.
4. Öffne:
   C:\Users\timot\perkiscreation-website
5. Mit Strg + V einfügen.
6. „Dateien im Ziel ersetzen“ auswählen.
7. Falls vorhanden: postcss.config.mjs löschen.
8. VS Code neu öffnen.
9. Terminal -> New Terminal.
10. Eingeben:
    npm run dev
11. Öffnen:
    http://localhost:3000
