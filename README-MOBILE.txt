EINBAU:
1. ZIP entpacken.
2. Den Inhalt in C:\Users\timot\perkiscreation-website kopieren.
3. „Dateien im Ziel ersetzen“ auswählen.
4. In VS Code:
   npm install
   npm run dev
5. Prüfen unter http://localhost:3000
