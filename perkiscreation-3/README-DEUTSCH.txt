PERKISCREATION – KORRIGIERTE MOBILE VERSION

KORRIGIERT:
- Hero-Bild wird auf dem Handy vollständig angezeigt
- keine englischen Formnamen mehr
- klarer Ablauf in vier Schritten
- Mobile-First-Layout
- Hamburger-Menü
- Kontaktformular
- Desktop-Ansicht bleibt zusätzlich erhalten

EINBAU:
1. ZIP entpacken.
2. Gesamten Inhalt kopieren.
3. In diesen Ordner einfügen:
   C:\Users\timot\perkiscreation-website\perkiscreation-3
4. Bei Windows „Dateien im Ziel ersetzen“ auswählen.
5. Falls vorhanden, postcss.config.mjs löschen.
6. In VS Code genau PERKISCREATION-3 öffnen.
7. Terminal:
   npm install
   npm run dev

LOKAL TESTEN:
http://localhost:3000

WICHTIG ZUR DOMAIN:
